# Social-Network
Built using Flask,Peewee and other python packages
## Python Package required to run this project
* Flask
* peewee
* Bcrypt
* LoginManager
* Flask Wtforms
* Six

## Run the project
`python app.py`

## Install package
Use `pip` or `easy_install`
